# n8n AI Workflows

Production-grade AI automation system built with n8n, multi-agent architecture, and RAG pipelines.

---

## 🔧 What this repo contains

- n8n automation workflows
- AI-powered agents
- RAG pipelines (Qdrant / Supabase)
- Telegram & API integrations

---

## 📂 Folder Structure

- workflows/ → automation JSON files  
- agents/ → AI logic  
- rag/ → knowledge base system  
- docs/ → documentation 

## 🧠 Architecture

- Data ingestion (RSS/API)
- Processing layer (n8n workflows)
- AI layer (OpenAI / agents)
- Memory layer (Qdrant / Supabase)
- Output layer (Telegram / Webhooks)
